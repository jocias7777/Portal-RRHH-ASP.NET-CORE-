# Sistema de Gestión de Empleados - Portal RRHH
Sistema de información para la gestión de recursos humanos desarrollado en ASP.NET Core 8.0.
## Características
- **Gestión de Personal**: Registro y mantenimiento de empleados
- **Control de Asistencia**: Registro y seguimiento de asistencia diaria
- **Gestión de Vacaciones**: Solicitud y aprobación de vacaciones
- **Nómina**: Cálculo y gestión de planilla
- **Prestaciones**: Gestión de bonificaciones y otras prestaciones
- **Reclutamiento**: Gestión de candidatos y plazas vacanttes
- **Evaluaciones**: Evaluaciones de desempeño del personal
## Tecnologías
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Bootstrap 5
- jQuery
## Requisitos
- .NET 8.0 SDK
- SQL Server 
## Instalación
1. Clonar el repositorio
2. Restaurar paquetes: `dotnet restore`
3. Ejecutar migraciones: `dotnet ef database update`
4. Iniciar: `dotnet run`
## Configuración
Editar `appsettings.json` para configurar la cadena de conexión.
